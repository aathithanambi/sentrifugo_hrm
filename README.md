Open source Hrm tool for office management

      MYSQL_HOST: db
      MYSQL_ROOT_USER: root
      MYSQL_ROOT_PASSWORD: root

      docker-compose up -d
      Complete the configuration at localhost:8080/
      
# Final step

      docker exec -it <docker id> bash
